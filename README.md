# Java 학습 저장소

Java 기초부터 실전까지 학습한 내용을 정리한 저장소입니다.

## 학습 과정

### 1. 기본편

#### 01. 김영한_자바 입문
Java의 기본 문법과 핵심 개념을 학습합니다.

**학습 내용:**
- Hello Java 및 주석 (comment)
- 변수 (variable)
- 연산자 (operator)
- 스코프 (scope)
- 형변환 (casting)
- Scanner 입력
- 배열 (array)
  - 1차원 배열
  - 2차원 배열
  - Enhanced for문
- 메서드 (method)
  - 메서드 정의와 호출
  - 매개변수와 반환값
  - 메서드 오버로딩 (overloading)

#### 02. 김영한_실전 자바
객체 지향 프로그래밍의 기초를 학습합니다.

**학습 내용:**
- 클래스 (class)
  - 클래스 기본 사용법
  - 객체 생성과 참조
- 참조 (reference)
  - 참조형 변수
  - 메서드와 참조
- 객체 지향 프로그래밍 (OOP)
  - 절차 지향에서 객체 지향으로
  - 데이터와 메서드의 캡슐화
- 생성자 (constructor)

## 프로젝트 구조

```
Java/
├── 1. 기본편/
│   ├── 01. 김영한_자바 입문/
│   │   └── src/
│   │       ├── variable/      # 변수 학습
│   │       ├── operator/      # 연산자 학습
│   │       ├── scope/         # 스코프 학습
│   │       ├── casting/       # 형변환 학습
│   │       ├── scanner/       # Scanner 학습
│   │       ├── array/         # 배열 학습
│   │       ├── method/        # 메서드 학습
│   │       └── ex/            # 연습 문제
│   └── 02. 김영한_실전 자바/
│       └── src/
│           ├── class1/        # 클래스 기초
│           ├── ref/           # 참조
│           ├── oop1/          # 객체 지향 프로그래밍
│           └── construct/     # 생성자
└── README.md
```

## 개발 환경

- Language: Java
- IDE: IntelliJ IDEA

## 학습 방법

각 주제별로 예제 코드를 직접 작성하고 실행하며 학습합니다.
연습 문제(`ex/` 폴더)를 통해 학습한 내용을 복습하고 적용합니다.
